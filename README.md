## ⚡ FLUJOS PRINCIPALES

### Flujo A: Nueva Aplicación (CEO → Market)
```
CEO idea → Gestor analiza → Loco propone tech → Desarrollo setup
→ Marketing pre-venta → Launch en 4 semanas
```

### Flujo B: Cliente contacta WhatsApp
```
Cliente → Evolution API → n8n → Admin analiza
→ auto-responde (con aprobación CEO) → Ticket si necesario
```

### Flujo C: Detecta oportunidad
```
Marketing ve trend → Gestor valida → Loco propone app
→ CEO decide → Nuevo proyecto
```

## ❓ PREGUNTAS FRECUENTES

**P: ¿Cuánto cuesta implementar?**
R: ~ARS 18.500/mes después del setup inicial (ARS 50K)

**P: ¿En cuánto tiempo se implementa?**
R: 30 días para funcionalidad básica, 6 meses para escala completa

**P: ¿Necesito desarrolladores?**
R: Sí, al menos 1 DevOps/Backend. Los agentes no reemplazan, complementan.

**P: ¿Qué pasa si un agente falla?**
R: Escalation automática a CEO + fallback a humano

**P: ¿Es realmente "100% automático"?**
R: ~85% automático. Las decisiones críticas requieren aprobación CEO.
